## 1. Environment Setup
Install iGibson

'''bash
git clone https://github.com/StanfordVL/iGibson --recursive
cd iGibson
conda create -y -n igibson python=3.8
conda activate igibson
pip install -e . # This step takes about 4 minutes
'''
